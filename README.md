# Simple Notes

シンプルに書いて、検索・タグ・アーカイブで整理できるブラウザ用メモ帳アプリです。

公開URL: https://simple-notes-drab.vercel.app/

## 概要

Simple Notes は、Next.js、TypeScript、Tailwind CSS で作った学習用のメモ帳アプリです。
メモの作成、編集、検索、タグ付け、ピン留め、アーカイブ、削除ができます。

保存にはブラウザの `localStorage` を使用しています。バックエンドやデータベースは使っていないため、フロントエンドだけで動く構成です。

## 主な機能

- メモの作成、編集、削除
- リッチテキストエディタによる本文編集
- Markdownショートカット入力による見出し、リスト、引用、コードブロックの作成
- メモのピン留め
- メモのアーカイブ、アーカイブから戻す操作
- 更新日、作成日、タイトルによる並び替え
- 複数メモの選択削除
- タイトル、本文、タグを対象にした検索
- タグの追加、削除、再利用
- 複数タグによる絞り込み
- メモ一覧の表示件数切り替え
- ライトモード / ダークモードの切り替え
- `localStorage` による自動保存
- 初期メモへのリセット
- UI確認用のサンプルメモ追加
- Markdown入力確認用のテストメモ追加

## 使い方

1. 左側の `+ 新規メモ` からメモを作成します。
2. 右側の編集エリアでタイトルと本文を入力します。
3. 上部のタグ欄からタグを追加できます。
4. 左側の検索欄で、タイトル・本文・タグをまとめて検索できます。
5. タグフィルターや並び替えを使って、メモ一覧を整理できます。
6. 不要だけど残しておきたいメモは、削除せずにアーカイブできます。

## Markdown入力

本文エリアでは、編集とプレビューを分けずにそのまま書けるリッチエディタを使っています。
行頭で以下のように入力してスペースを押すと、見出しやリストなどのブロックへ変換されます。

````txt
#   見出し1
##  見出し2
### 見出し3
-   箇条書き
1.  番号付きリスト
>   引用
``` コードブロック
````

## 保存について

メモはブラウザの `localStorage` に保存されます。
同じブラウザであれば、ページを再読み込みしてもメモは残ります。

ただし、別の端末や別のブラウザとは同期されません。また、ブラウザのデータを削除すると保存したメモも消える可能性があります。
