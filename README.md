# Simple Notes

シンプルに書いて、検索・タグ・アーカイブ・ToDoで整理できるブラウザ用メモアプリです。

公開URL: https://simple-notes-drab.vercel.app/

## 概要

Simple Notes は、Next.js、TypeScript、Tailwind CSSで作った学習用アプリです。
メモとToDoを画面上部から切り替え、それぞれを独立して整理できます。

データはブラウザの `localStorage` に保存します。バックエンドやデータベースを使わず、フロントエンドの基本的な状態管理やデータ保存を学びやすい構成にしています。

## 主な機能

### メモ

- メモの作成、編集、削除
- リッチテキストエディタによる本文編集
- Markdownショートカットによる見出し、リスト、引用、コードブロックの作成
- タイトル、本文、タグを対象にした検索
- 複数タグの追加、削除、再利用、絞り込み
- ピン留め
- アーカイブと復元
- 複数メモの選択削除
- 更新日、作成日、タイトルによる並び替え
- メモ一覧の折りたたみ

### ToDo

- ToDoリストの作成、選択、削除
- リストごとのタスク追加、完了切り替え、削除
- すべて、未完了、完了によるタスク絞り込み
- リスト名、タグ、タスク名を対象にした検索
- リストへの複数タグの追加、削除、再利用、絞り込み
- リストのピン留め
- リストのアーカイブと復元
- 複数リストの選択、アーカイブ、削除
- 更新日、作成日、タイトルによる並び替え
- リスト一覧の折りたたみ

### 共通

- メモ / ToDo の画面切り替え
- PC、スマートフォンに対応したレスポンシブUI
- スマートフォン用の展開式操作メニュー
- ライト、ダーク、端末設定に合わせるテーマ切り替え
- ユーザー名の変更
- `localStorage` による自動保存
- 初期データへのリセット
- 開発・UI確認用のサンプルデータ追加

## 使い方

### メモを書く

1. 上部の切り替えで `メモ` を選択します。
2. サイドバーの `+ 新規メモ` からメモを作成します。
3. 編集エリアでタイトルと本文を入力します。
4. 上部のタグ欄から新しいタグや過去に使ったタグを追加できます。
5. 検索、タグフィルター、並び替えを使ってメモを整理します。

### ToDoを管理する

1. 上部の切り替えで `ToDo` を選択します。
2. サイドバーの `+ 新規リスト` からリストを作成します。
3. 選択したリストへタスクを追加します。
4. 完了したタスクにチェックを入れます。
5. リストにはタグ、ピン留め、アーカイブを設定できます。

PCでは一覧項目にカーソルを合わせると操作アイコンが表示されます。スマートフォンでは `…` ボタンから操作メニューを開きます。

## Markdown入力

メモ本文では、編集とプレビューを分けずに書けるリッチテキストエディタを使っています。
行頭で次の記号を入力してスペースを押すと、対応するブロックへ変換されます。

````txt
#   見出し1
##  見出し2
### 見出し3
-   箇条書き
1.  番号付きリスト
>   引用
``` コードブロック
````

## ローカルでの起動

```bash
npm install
npm run dev
```

ブラウザで以下を開きます。

```txt
http://localhost:3000
```

```bash
npm run dev -- --port 3001
```

## 保存について

メモ、ToDoリスト、タスク、タグ、ユーザー名、テーマ設定はブラウザの `localStorage` に保存されます。同じブラウザであれば、ページを再読み込みしてもデータは残ります。

別の端末や別のブラウザとは同期されません。また、ブラウザのサイトデータを削除すると保存内容が消える可能性があります。
